# タスク登録

## 概要

タスクに情報を登録、同時にタスクステータスに初期状態で登録

## 入力データ

| 和名        | 物理名      | 型     | 値域 | 必須 |
| ----------- | ----------- | ------ | ---- | ---- |
| タイトル    | title       | 文字列 | -    | 〇   |
| テキスト    | description | 文字列 | -    | -    |
| 優先度      | priority    | 数値   | -    | -    |
| 期限        | due_date    | 日時   | -    | -    |
| ユーザー ID | userId      | 文字列 | -    | 〇   |

## 出力データ

| 和名      | 物理名 | 型     | 値域       | 必須 |
| --------- | ------ | ------ | ---------- | ---- |
| タスク ID | taskId | 文字列 | -          | 〇   |
| 処理結果  | result | 論理   | true,false | 〇   |
