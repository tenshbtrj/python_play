# API 一覧

| API 名       | 機能概要                                                   |
| ------------ | ---------------------------------------------------------- |
| タスク検索   | タスクとタスクステータスを結合し、ユーザー ID で検索       |
| タスク登録   | タスクに情報を登録、同時にタスクステータスに初期状態で登録 |
| タスク更新   | タスクステータスを更新                                     |
| ユーザー検索 | ユーザー ID とパスワードで一致するユーザを検索             |
| ユーザー登録 | ユーザーに登録                                             |
